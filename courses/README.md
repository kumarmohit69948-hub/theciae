# Course Resource Folders

Folder structure for the B.Tech. Agricultural Engineering programme,
generated from the ICAR 6th Deans' Committee syllabus
([assets/btech-agril-engg-6th-dean-syllabus.pdf](../assets/btech-agril-engg-6th-dean-syllabus.pdf)).

Each course folder contains one subfolder per lecture (from the syllabus
"Planning of Lectures" tables) — drop notes, papers and other resources
into the matching lecture folder.
