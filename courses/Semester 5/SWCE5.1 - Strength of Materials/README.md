# SWCE5.1 — Strength of Materials

B.Tech. Agricultural Engineering (ICAR 6th Deans' Committee syllabus).

## Lecture plan

- **Lecture 1:** Introduction to strength of materials
- **Lecture 2:** Slope and deflection of beam using integration techniques
- **Lecture 3:** Moment area theorems
- **Lecture 4:** Problems of slope and deflection
- **Lecture 5:** Theory of columns and struts
- **Lecture 6:** Problems of column and struts
- **Lecture 7:** Analysis of rivet connections
- **Lecture 9:** Stability analysis of masonry dam
- **Lecture 10:** Problems on masonry dam
- **Lecture 11:** Statically indeterminate structures- analysis of propped beams
- **Lecture 12:** Analysis of fixed beams
- **Lecture 13:** Analysis of continuous beams using superimposition and three-moment equation
- **Lecture 14:** Analysis of beam using moment distribution method and solving problems
